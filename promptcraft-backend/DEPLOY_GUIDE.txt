# PromptCraft — Deploy Guide 🚀
# Sirf copy-paste karo, 15 minute mein live!

## Kya Chahiye:
- GitHub account (free) → github.com
- Vercel account (free) → vercel.com
- Groq API key (free) → console.groq.com

---

## STEP 1 — Groq API Key Lo (2 min)
1. console.groq.com pe jaao
2. Sign up (Google se bhi ho sakta hai)
3. "API Keys" → "Create API Key"
4. Key copy karke safe jagah rakh lo (gsk_... se shuru hogi)

---

## STEP 2 — GitHub Pe Upload Karo (5 min)
1. github.com pe jaao → Sign up / Login
2. "New Repository" button dabao
3. Name do: "promptcraft"
4. "Public" select karo
5. "Create repository" dabao
6. "uploading an existing file" link pe click karo
7. Saari files drag & drop karo:
   - api/generate.js
   - public/index.html
   - public/style.css
   - public/script.js
   - package.json
   - vercel.json
   - .env.example
8. "Commit changes" dabao ✅

---

## STEP 3 — Vercel Pe Deploy Karo (5 min)
1. vercel.com pe jaao → Sign up with GitHub
2. "Add New Project" dabao
3. "Import" → apna "promptcraft" repo select karo
4. "Deploy" dabao

---

## STEP 4 — API Key Add Karo (2 min) ⚠️ MOST IMPORTANT
1. Vercel dashboard mein project kholo
2. "Settings" tab pe jaao
3. "Environment Variables" section mein jaao
4. Add karo:
   - NAME:  GROQ_API_KEY
   - VALUE: gsk_tumhari_key_yahan
5. "Save" dabao
6. "Deployments" → "Redeploy" dabao

---

## STEP 5 — Live! 🎉
Vercel tumhe ek free URL dega jaise:
https://promptcraft-abc123.vercel.app

Ye URL sabke liye kaam karega — koi API key nahi mangegi!

---

## Apna Domain Add Karna (Optional)
1. Vercel → Settings → Domains
2. Apna domain add karo (jaise promptcraft.in)
3. DNS settings update karo

---

## Koi Problem Aaye?
- Vercel logs check karo: Project → Functions → Logs
- Error "API key missing": Step 4 dobara karo
- Error "Module not found": package.json check karo
